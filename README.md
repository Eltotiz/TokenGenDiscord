# TokenGenDiscord
TokenGen es una herramienta que permite generar tokens de usuarios de discord NO chequeados y te los guarda ordenadamente en un archivo txt.

# Velocidad
100 tokens/5s

# Requisitos

python 3.x

base64

# Instalacion e ejecucion

pip install -r requirements.txt

python3 main.py
